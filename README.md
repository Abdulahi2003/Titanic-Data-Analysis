# Titanic-Data-Analysis
End-to-end data analysis project using SQL, Python and Tableau

## Overview

This project analyzes the Titanic dataset to understand what factors affected passenger survival. The goal was to see if survival rate had anything to do with passenger class. Also, assess the age of the passengers within each class to identify vulnerable age categories. It covers data cleaning, feature creation, and visualization.

## Tools Used

* Python (Pandas)
* SQL
* Microsoft Excel
* Tableau

## Process

* Queried data using SQL
* Cleaned and processed data in Python (handled missing values, formatted columns)
* Created a new feature: age categories (Children, Youth & Adults, Seniors)
* Exported the cleaned dataset to Excel
* Built a dashboard in Tableau

## Key Insights

* Children had higher survival rates
* First-class passengers were more likely to survive
* Most passengers were in the Youth & Adults category
* Survival varied across different groups
